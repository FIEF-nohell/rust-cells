# Learnings

Append-only lessons from past sessions. Filename format: `YYYY-MM-DD-<short-slug>.md`. Frontmatter required: `date`, `tags`, `severity`, `applies-to`. Never delete a learning; supersede an old one by writing a newer file and adding a `superseded-by:` line to the old one's frontmatter. Write a learning only when it would change behavior next time; zero per session is fine.
